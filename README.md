# VIM commands and shorthand

### Open/View commands

from shell:

`vim`   -> open vim

`vim .` -> open a list of directories and files of your current position

`vim src/hello.js` -> open `hello.js` file

### Edit/write commands

from VIM:

`:e` -> edit

`:e hello.js` -> edit `hello.js` file

`:x` -> save a file and quit

`:q!` -> quit without saving
